# Prashanth C - Portfolio Website

A professional portfolio website showcasing education, technical skills, projects, and achievements in AI and Data Science.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Styling**: Clean and professional design with gradient backgrounds
- **Sections**:
  - Hero section with contact information
  - Technical skills categorized by type
  - Featured projects with descriptions and tools used
  - Key achievements and accomplishments
  - Contact information
